# Product Decisions

## V1 Scope Decisions

| Question | Decision |
|-----------|-----------|
| Testimonials | Do not include testimonials in V1 |
| Blog | Included in V1 |
| Sample chapters | Placeholder samples acceptable for V1 |
| Future products | `/future` page with "Coming Soon" cards |
| Primary CTA | View Sample Chapters |
| Secondary CTA | Request a Book |
| Dynamic routes | Not in Phase 1 |
| Forms | Real Formspree submission to `https://formspree.io/f/mrewzvoz` |
| AI tutoring | Not planned |

## Reasoning

These decisions reduce scope and allow Ganitra V1 to focus on validating interest in the textbook series and collecting leads.
